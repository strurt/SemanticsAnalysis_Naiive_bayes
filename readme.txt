Semantic Analysis Using Naive Bayes

Dataset-> IMDB Movie Review Datset 
Link : https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

Steps : 

Data Pre-Processing -> remove html tags, remove special characters, remove stop words , stemming

Vectorisation / Bag of words -> count the frequency of each occurence of words in given review; each unqiue word in the dataset will be a column 

Train-Test Split 

Apply Naive Bayes to get the probabilty of the label either being positve or negative
